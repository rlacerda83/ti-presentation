Cálculo de Idade - Exemplo Prático

Como Funciona

Nossa Necessidade: Calcular a idade de uma pessoa baseada no seu ano de nascimento.

Abaixo descrevemos essa pequena aplicação com um pseudo código e com o código nativo, no caso javascript.


Pseudo Código

Ano Atual = 2017
Ano de Nascimento Informado = Informado pelo usuário
Se o usuário não informar o ano de nascimento:
    Cálculo cancelado
Fim da Condição Se

Idade = Ano Atual - Ano de Nascimento Informado
Mostra Resultado

Código Nativo

function calculateAge() {
    let currentYear = new Date().getFullYear();
    let userBirthdayear = $('#year').val();
    if (!userBirthdayear) {
        $('#userAge').focus();
        return false;
    }
    $('#userAge').val(currentYear - userBirthdayear);
    $('#myModal').modal();
}
